Jak zkrotit kecálka. tool_progress, approvals, reasoning & quiet mode

Hermes Agent · Config · Quality of Life

Config (natrvalo)

Uprav ~/.hermes/config.yaml nebo použij hermes config set:

display:
  tool_progress: none    # schová "čtu soubor, zapisuju"
  show_reasoning: false  # schová reasoning trace
approvals:
  mode: smart            # allow/deny jen u rizikových operací

Nebo přes CLI:

hermes config set display.tool_progress none
hermes config set display.show_reasoning false
hermes config set approvals.mode smart

⚠️ Po změně configu restartovat gateway: hermes gateway restart

Slash commandy (jen CLI)

  • /verbose — cykluje: off → new → all → verbose
  • /quiet (alias -Q) — potlačí banner, spinner, tool previews
  • /yolo — vypne allow/deny prompty úplně

Flagy při spuštění

hermes -Q                     # quiet režim
hermes --yolo                 # bez potvrzování
hermes chat -q "otázka" -Q   # jednorázovka potichu

Co se stane

  • ❌ zmizí: "Reading file…", "Writing to…", "Running command…"
  • ❌ zmizí: reasoning / thinking bloky
  • ✅ zůstane: finální odpověď, chyby,高危 warningy
